- 实践与认识
	- 科学实践观
		- 内涵 = 实践本质
		- 意义 【唯反历改/违反立改】
			1. 【唯物】客服旧唯物主义根本缺陷，奠定辩证唯物主义
			2. 【反映】建立科学+能动+革命の反映论
			3. 【历史】解释社会生活の实践本质，奠定科学历史观
			4. 【改造】为能动认识改造世界提供思想方法+工作方法
	- 实践 【本特构多认/bentgo多人】
		- what 本质 = 人类改造世界の社会性の物质活动【改社物】
		- what 基本特征 【客自社】
			1. 客观实在性
			2. 自觉能动性
			3. 社会历史性
		- 基本结构 【构系变】
			- what 基本结构
				- 实践主体：个人主体/群体主体/人类主体
				- 实践客体：天然/人工+自然/社会+物质性/精神性
				- 实践中介：物质性工具+语言符号工具
			- what 主客体关系
				- 实践关系 【**最根本**】
				- 认识关系
				- 价值关系
				- 实践主客体关系 $\xlongequal{本质上}$ 认识主客体关系
			- what 结构变化
				- 主体客体化 (创造)
				- 客体主体化 (延伸)
		- what 形式多样性 【物政科虚】
			- 物质生产
			- 社会政治
			- 科学文化
			- **虚拟实践**
				- 实质 = 主客体 【主客数虚双】
					1. 通过数字化中介
					2. 在虚拟空间 
					3. 双向对象化活动
				- 性质 【交开间】
					- 交互性
					- 开放性
					- 间接性
		- why 对认识の决定作用 = 认识の基础 【基源动目准】
			1. = 认识の来源
			2. = 认识发展の动力
			3. = 认识の目的
			4. = 检验认识真理性の唯一标准
	- 认识
		- 本质
			0. 唯心先验，唯物反映
			1. 主观唯心：主观自生，生而知之
			2. 客观唯心：上帝启示/客观精神产物
			3. 旧唯物：直观，消极+被动の反映论
			4. 辩证唯物：实践+辩证+能动の反映论
		- 实践到认识 【第一次飞跃】
			- 感性认识 = 实践基础上+感觉器官直接+事物现象/外部联系/各个方面の认识
				- 感觉
				- 知觉
				- 表象
			- 理性认识 = 借助抽象思维+概括整理大量感性材料+达到关于事物本质/全体/内部联系/自身规律性の认识
				- 概念
				- 判断
				- 推理
			- what 关系 【赖深渗】
				1. 感性认识 $\xleftarrow{依赖}$ 理性认识 
				2. 感性认识 $\xrightarrow{有待发展深化成}$ 理性认识
				3. 感性认识 $\xleftrightarrow{相互渗透相互包含}$ 理性认识
			- how 感上升理基本条件 【投查理科】
				1. 投身实践，深入调查
				2. 发挥思考，运用理论思维&科学抽象
		- 认识到实践 【第二次飞跃】
			- why 重要必要性
				1. 【实践角度】认识世界目的是改造世界
				2. 【认识角度】认识的真理性只有实践能检验发展
			- how 中介环节
				1. 明确目的
				2. 形成理念
				3. 制定计划
				4. 中间试验
				5. 运用科学实践法
				6. 对人民群众组织宣传
	- 实践+认识
		- 辩证运动过程 【飞再】
			1. 感性认识 $\to$ 理性认识 $\to$ 实践
			2. 实践 认识 再实践 再认识
		- 辩证运动规律 【螺完统】
			1. 认识反复循环无限发展，但波浪前进螺旋上升
			2. 完成 = 具体事物认识 + 没完成 = 实践认识の运动过程
			3. 遵循具体&历史的统一
				1. 具体统一：主观认识符合具体时/地/条下客观实践
				2. 历史统一：主观认识符合历史发展阶段の客观实践
- 真理与价值
	- 真理
		- 客观性 = 真理内容是对客观事物规律の正确反映 【客规正反】
			- 一元性 = 同条件+特定认识客体+真理性认知只有一个  【条客真一】
				- 客观性 $\xrightarrow{决定}$ 一元性
				- 真理一元 价值多元
			- 形式主观性
		- 绝对性 = 真理主客观统一の确定性 + 发展无限性 【统确发限】
		- 相对性 = 对客观事物本质&发展规律の正确认识 有限度&不完善 【正认限善】
		- 绝对性相对性辩证统一
			1. 【真理两重性】
				1. 两者相互依存 = 即绝对又相对
				2. 两者相互包含 = 绝对有相对，相对有绝对
			2. 【真理发展】不断深化，相对性 $\to$ 绝对性
		- 谬误
			1. 真理谬误相互对立
			2. 真理谬误对立相对，可互相转化
		- 检验标准 = 实践
			- why
				1. 【真理本性角度】真理本性 = 主客观符合
				2. 【实践特点角度】实践具有直接现实性
			- 实践标准 = 确定性+不确定性统一
				- 确定性 = 绝对性 = 实践作为检验真理标准の唯一性+归根到底性+最终性【唯根终】
				- 不确定性 = 条件性
					- 主客观条件制约
					- 历史条件制约
	- 价值 = 客体对人/群体/社会の意义
		- 基本特性 【主客多历】
			1. 主体性
			2. 客观性
			3. 多维性
			4. 社会历史性
		- 评价特点 【关直确科/官职check】
			1. 以 主客体价值关系 为 认识对象
			2. 结果与主体直接相关
			3. 结果正确性依赖对客体状况&主体需要の认识
			4. 有科学&非科学之别
		- 价值观
	- 真理+价值
		- 真理尺度 = 实践中 必须遵循の 反映客观事物本质规律の 真理
			- 体现 **规律性**
		- 价值尺度 = 实践中 按照自己尺度 去认识世界+改造世界 
			- 体现 **目的性**
		- 真理尺度+价值尺度の关系 【规辩历相】
			1. 任何实践在两种尺度共同制约下进行，合规律性+合目的性统一
			2. 紧密联系，不可分割の辩证统一关系
				1. 价值尺度 $\xleftarrow{前提}$ 真理尺度
				2. 真理尺度 $\xleftarrow{主体意义}$ 价值尺度
			3. 具体的，历史的
			4. 相互制约，相互引导，相互促进
- 认识世界改造世界
	- 认识世界+改造世界
		- 认识世界 = 能动反映客体
		- 改造世界 = 改变事物现存形式
		- 辩证统一
			1. 认识世界 $\xrightarrow{有助于}$ 改造世界
			2. 改造世界实践中才深化拓展正确认识
		- 矛盾过程 = 世界不自动满足人，人不满足现存形式
		- 主观世界+客观世界
			- 辩证统一 = 经典左有助于右+右有助于左+相辅相成
			- 马克思主义在...的意义
				1. 提高认识世界の能力
				2. 提高思想修养+精神境界
				3. 改造外部世界时，重视+努力人自身改造
		- 必然+自由
			- 自由の条件 = 
				1. 认识条件 = 客观事物正确认识
				2. 实践条件 = 规律性认识指导实践
			- 认识世界改造世界根本目标 =  认识必然+争取自由
	- 实事求是
		1. 一切从实际出发是马克思主义认识论の根本要求
		2. 实事求是是党思想路线の核心
	- 守正创新 【坚理实】
		1. 坚持守正创新 【摇停统】
			1. 坚持守正不动摇
			2. 坚持创新不停步
			3. 坚持守正+创新辩证统一
		2. 努力实现理论创新+实践创新の良性互动 【源南互】
			1. 实践创新为理论创新提供不竭动力源泉
			2. 理论创新为实践创新提供科学行动指南
			3. 良性互动，相互激发，共同促进
