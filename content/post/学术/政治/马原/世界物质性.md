1. 世界多样&物质统一
	1. 哲学基本问题
		1. 第一性问题 物质意识决定 唯心/唯物
		2. 同一性问题  可知/不可知
	2. 物质 【范存重】
		1. 物质
			1. 不依赖人类意识存在
			2. 有能被人类意识反映の客观实在性 【**唯一特性**】
		2. 物质范畴
			1. what
				1. 高度抽象哲学概念
				2. 世界客观存在事物共同本质概括
			2. why
				1. ✅唯物一元论                 ❌唯心一元二元论
				2. ✅能动反映论&可知论 ❌不可知论
				3. ✅唯物论&辩证法          ❌形而上学唯物主义缺陷
				4. 唯物主义自然观&历史观统一 $\Rightarrow$ 奠基彻底唯物主义
		3. 物质存在方式 what
			1. 物质の根本属性 = 运动
			2. 物质运动是绝对的，但存在相对静止
			3. 时间&空间 是 运动物质の基本存在形式
		4. 世界物质二重化
			1. 关键/原因 = 人类改造世界の实践活动
			2. 自然界 $\xrightarrow{分化出}$ 人类社会
			3. 客观世界 $\xrightarrow{分化出}$ 主观世界
	3. 物质&意识
		1. 物质决定意识
			1. 意识本质 what
				1. 人脑の机能&属性
				2. 客观世界映象
			2. 决定于
				1. 起源角度：意识 = 自然界长期发展产物+社会历史发展产物
				2. 本质角度：本质what
		2. 意识对物质有反作用 = 能动作用【目计创性，导践行理】
			1. 意识有目的性&计划性
			2. 意识有创造性
			3. 意识有指导实践改造客观世界の作用
			4. 意识有调控人行为&生理活动の作用
		3. 客观规律&主观能动统一
			1. 尊重客观规律 是 发挥主观能动性 の前提
			2. 发挥主观能动性 才能 认识利用客观规律 how
				1. 前提：从实际出发
				2. 根本途径：实践
				3. 依赖：物质条件&物质手段
			3. 实践是客观规律性+主观能动性统一基础
		4. 意识&AI
			1. AI what = 人意识能动性の特殊表现
			2. why  【统社语化】
				1. 【统一体】
					✅ 人类意识是知情意统一体
					❌ AI 不具备感情信念意志的人类意识形态
				2. 【社会】社会性是人意识の固有属性
				3. 【语言】自然语言 是 思维物质外壳&意识物质形式
				4. 【化约】人脑有很多内容无法被化约为数字信号 (潜意识)
	4. 世界の物质统一性
		1. 世界统一性在于它的物质性
			1. 【自然】自然界 是物质的
			2. 【社会】人类社会 本质上是生产实践基础形成の物质体系
			3. 【人】人的意识统一于物质
		2. 世界の物质统一性是多样性的统一
2. 普遍联系&变化发展
	1. 联系&发展
		1. 联系 what 【客普多条】
			1. 客观性
			2. 普遍性
			3. 多样性
			4. 条件性
		2. 变化 what = 一切改变
		3. 发展 what = 变化中前进、上升の运动
			发展实质 = 新事物产生&旧事物灭亡
		4. 基本环节 【内本因必实】
			1. 内容与形式
			2. 本质与表象
			3. 原因与结果
			4. 必然与偶然
			5. 现实与可能
	2. 矛盾 (对立统一规律)
		1. 同一斗争
			1. 同一性 【前吸转/千禧传】
				1. 事物存在和发展的**前**提
				2. 矛盾双方**吸**收对自身有利因素，相互作用中各自得到发展
				3. 规定事物の**转**化可能和发展趋势
			2. 斗争性 【力变统一体】
				1. 【各自】矛盾双方促进矛盾双方力量变化，造成力量不平衡，为对立面转化&事物质变创造条件
				2. 【总体】矛盾双方的斗争是一种矛盾统一体向另一种矛盾统一体过渡的决定性力量
			3. 同一+斗争
				1. why 【结合地位】
					1. 矛盾同一性和斗争性相互结合，共同作用
					2. 不同条件下，两者地位有所不同
				2. how  【结把谐争】
					1. 善于结合两者，一者中把握另一者
					2. 正确把握和谐对事物发展的作用
					3. 大力发扬斗争精神
		2. 普遍特殊
			1. 普遍性
				1. 【静态】矛盾存在于一切事物中
				2. 【动态】存在于一切事物发展过程始终
				3. 【自身】旧矛盾解决，新矛盾产生
			2. 特殊性
				1. 各个具体事物的矛盾
				2. 各个矛盾的各个方面
				3. 在发展不同阶段
				4. 各有不同特点
			3. 普遍+特殊 = 辩证统一关系
				1. 普遍性=共性，特殊性=个性，共性无条件绝对，个性有条件相对
				2. 现实存在事物的矛盾都是共性&个性有机统一
		3. 主次矛盾
			1. what
				1. 主要矛盾 = 矛盾体系处支配地位，对事物发展起决定作用
				2. 次要矛盾 = 矛盾体系处从属地位，对事物发展起次要作用
			2. how = 坚持"重点论"&"两点论"
	3. 量变质变规律
		1. what
			1. 质 = 事物区别于他物の内在规定性
			2. 量 = 可以用数量关系表示の规定性
			3. 度 = 保持稳定性の数量界限
			4. 量变 = 
				1. 事物数量增减&组成要素排列次序变动
				2. 保持事物相对稳定性の不显著变化
				3. 体现事物发展渐进过程连续性
			5. 质变 = 
				1. 事物性质根本变化
				2. 事物质态向质态の飞跃
				3. 体现事物发展渐进过程连续性の中断
		2. why 质变&量变关系
			1. 量变是质变必然准备
			2. 质变是量变必然结果，为新量变开辟道路
			3. 量变质变相互渗透
		3. how 方法论
			1. 平日踏实量变
			2. 质变抓住机遇
	4. 否定之否定规律 【否否辩阶曲意】
		1. what 否定之否定 【两支高】
			1. 任何事物内部都有肯有否
			2. 矛盾双方作用，否定上升支配定位，事物从肯走向自否
			3. 再由否走向更高阶段の否
		2. what 辩证否定观 【自发联扬】
			1. 否定 = 事物的自我否定，自我发展，事物内部矛盾の结果
			2. 否定 = 事物发展の环节，旧事物向新事物の转变，旧质向新质の飞跃
			3. 否定 = 新旧事物联系の环节，旧事物孕育新事物
			4. 辩证否定实质 = "扬弃"，新事物对旧事物批判&继承，克消保积
		3. what 辩证发展过程三个阶段
			1. 肯-否-否の否
			2. 周期性，波浪前进，螺旋上升
		4. what 前进性&曲折性
			1. 前进性 = 每一次否定都是质变，每一周期都开放，无不可否定の终点
			2. 曲折性 = 回复性，有时停顿甚至倒退
		5. how 指导意义
			1. 树立辩证否定观，no 形而上学/肯否一切，采取科学分析态度
			2. 正确看待事物发展过程，看到曲折 & 看到前途光明
3. 唯物辩证法
