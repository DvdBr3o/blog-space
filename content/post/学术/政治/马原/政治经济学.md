- 商品
	- 自然经济 = 自给自足+使用价值目的
	- 商品经济
		1. 含义 = 交换目的+自然经济对立物
		2. 社会历史条件
			1. 社会分工
			2. 生产资料&劳动产品属于不同所有者
		3. 发展阶段
			1. 出现于原始社会末期
			2. 奴隶社会封建社会发展，不主导
			3. 资本主义社会成为普遍经济形势
	- 二因素二重性
		- 商品二因素
			1. 使用价值 = 有用性
				1. 反映人与自然物质关系
				2. 商品自然属性
				3. 劳动产品共有属性
			2. 价值 = 无差别一般人类劳动
				1. = 人脑力体力耗费
				2. 商品*特有* **社会属性**
				3. 本质 = 生产者社会关系 = 物质利益关系
			3. 使用价值 vs 价值 の 对立统一
				1. 对立性 = 不可兼得
				2. 统一性 = 商品同时具有二因素
					1. 使用价值 $\xrightarrow{物质承担者}$ 价值
					2. 价值 $\xrightarrow{寓于}$ 使用价值 
		- 劳动二重性
			1. 具体劳动 = 生产一定使用价值の具体形式劳动
				1. 创造商品の使用价值
			2. 抽象劳动 = 撇开具体形式，无差别一般人类劳动 = 脑力体力耗费
				1. 形成商品の价值
			3. 具体劳动 vs 抽象劳动 の 对立统一
				1. 统一 = 
					1. 商品生产者在同一劳动过程不可分割の两方面
					2. 时间空间上统一
				2. 对立 = 
					1. 具体劳动 反映 劳动自然属性
					2. 抽象劳动 反映 劳动社会属性 = 商品生产者社会关系
	- 价值量决定
		1. 社会必要劳动时间 $\xrightarrow{决定}$ 商品价值量

		2. 社会必要劳动时间 随 劳动生产率 变化而变化 $$价值量 = k\frac{社会必要劳动时间}{劳动生产率}$$

		3. 价值量与简单劳动+复杂劳动有密切关系
			1. 以简单劳动为尺度
			2. 复杂劳动 = 自乘/多倍简单劳动
				(少量复杂劳动 = 多量简单劳动)
			3. 复杂简单劳动换算在商品交换过程自发实现
	- 价值形式发展
		1. 简单偶然价值形式
		2. 总和扩大价值形式
		3. 一般价值形式
		4. 货币形式
	- 货币
		- 本质 =  固定充当一般等价物の商品
		- **职能** 【价流贮付世】
			1. 价值尺度
			2. 流通手段
			3. 贮藏手段
			4. 支付手段
			5. 世界货币
- 价值规律
	- 主观内容+客观要求
		1. 价值量 $\xleftarrow{决定}$ 社会必要劳动时间
		2. 商品交换 $\xleftarrow{基础}$ 价值量
		3. 商品交换 $\xleftarrow{原则}$ 等价交换
	- 表现形式 = 价格围绕价值自发波动
	- 作用
		- 积极 【料产收】
			1. 自发调节生产资料+劳动力在社会生产部门分配比例
			2. 自发刺激社会生产力发展
			3. 自发调节社会收入分配
		- 消极 【费垄极】
			1. 自发性+滞后性 $\xrightarrow{可能导致}$ 社会资源浪费
			2. 导致垄断，阻碍技术进步
			3. 收入两极化
- 私有制商品经济

- 资本主义经济制度
	- 产生
		- 原因 = 生产力生产关系矛盾 $\xrightarrow{引起}$ 农村自然经济+城市行会组织瓦解 + 城乡资本主义生产关系产生
		- 途径
			1. 小商品经济分化
			2. 商人/高利贷者转化
		- 确立
			1. 【政治】资产阶级革命
			2. 【阶级】新兴资产阶级暴力夺权
			3. 【科技】工业革命
	- 原始积累
		- what 定义 = 
			1. 生产者生产资料分离
			2. 资本迅速集中于少数
			3. 资本主义迅速发展の历史过程
		- 途径
			1. 暴力手段剥夺农民土地
			2. 暴力手段掠夺货币财富
	- 劳动力
		- what 定义 = 人的劳动能力 = 人脑力+体力总和
		- 成为商品の基本条件
			1. 劳动者是法律上の自由人，能把劳动力当商品支配
			2. 劳动者没有生产资料+生活资料来源，依靠出卖劳动力为生
		- 劳动力商品特点 = 价值+使用价值の不同
			1. 价值
				1. 维持劳动者本人生存必需の生活资料价值
				2. 维持劳动者家属生存必需の生活资料价值
				3. 劳动者接受教育+训练支出费用
				4. 最低界限 $\xleftarrow{界定}$ 生活不可缺少生活资料价值
			2. 使用价值 = 价值源泉
		- 货币转化为资本の条件 = 货币购买劳动力商品带来剩余价值
	- 剩余价值
		1. 剩余价值生产
			1. 剩余价值生产过程 = 劳动过程+价格增殖过程统一
				1. 劳动过程三个基本要素=劳动+劳动对象+劳动资料
			2. 剩余价值生产の资本作用 = 生产资料 / 劳动力
				1. 不变资本 $c$ = 生产资料形式存在资本，改变物质形态，不发生价格增殖
				2. 可变资本 $v$ = 购买劳动力の资本，工人创造新价值，发生价格增殖
				3. 雇佣劳动者の剩余劳动 = 剩余价值の唯一源泉
			3. 剩余价值率 $m'=\frac{剩余价值}{可变资本}=\frac{剩余劳动}{必要劳动}=\frac{剩余劳动时间}{必要劳动时间}$
			4. 剩余价值生产基本方法
				1. 绝对剩余价值生产 = 必要劳动时间不变+工作日绝对延长+提高劳动强度
				2. 相对剩余价值生产 = 缩短必要劳动时间+工作日不变+延长剩余劳动时间
					1. 缩短必要劳动时间$\xleftarrow{依靠}$全社会劳动生产率提高
					2. 全社会劳动生产率提高 $\xleftarrow{导致}$ 资本家追逐超额剩余价值
		2. 资本积累 = 剩余价值资本化
			1. 资本主义再生产
				1. 简单再生产 = 剩余价值用于个人消费，生产规模不变
				2. 扩大再生产 = 剩余价值转化为资本，追加生产资料+劳动力，生产规模扩大
				3. 扩大再生产 = 资本主义再生产特点
			2. 源泉 = 剩余价值
			3. 规模取决于
				1. 资本家对工人剥削程度
				2. 劳动生产率高低
				3. 所用资本 $-$ 所费资本
				4. 资本家预付资本大小
			4. 资本构成
				1. 技术构成 = 生产技术决定の生产资料:劳动力
				2. 价值构成 = 不变资本:可变资本
				3. 有机构成 = 技术构成决定反应技术构成变化の价值构成
				4. 相对过剩人口 = 流动/潜在/停滞
		3. 资本循环周转
			1. 资本循环
				1. 阶段 = 购买 - 生产 - 售卖
				2. 职能 (对应阶段)
					1. 货币资本
					2. 生产资本
					3. 商品资本
				3. 前提条件
					1. 三种职能形式 空间并存
					2. 三种职能形式 时间继起
			2. 资本周转
				1. 含义 = 周而复始+不断反复の资本循环
				2. 因素
					1. 资本周转时间
					2. 生产资本の固定资本:流动资本の构成
				3. 制约 = 经济制度
			3. 社会总产品 = 社会总价值
				1. 价值形态 = 
				2. 物质形态 = 生产资料 + 消费资料
			4. 社会再生产
		4. 剩余价值分配
			1. 资本主义工资
				1. 本质 = 劳动力价值/价格
				2. 形式 = 计时/计件/血汗工资
			2. 剩余价值 $\xrightarrow{转化}$ 利润
				1. 利润 = 剩余价值观念上作为全部预付资本产物/增加额
				2. 剩余资本 $\xrightarrow{转化}$ 资本，掩盖资本家剥削
				3. 资本主义生产目的 = 获得利润
					不同部门资本家按等量资本要求等量利润原则瓜分剩余价值
				4. 平均利润率 = $\frac{剩余价值总量}{社会总资本}$
				5. 平均利润 = 按平均利润率计算获得利润
			3. 利润 $\xrightarrow{转化}$ 平均利润
				1. = 剩余价值规律+竞争规律の必然结果
				2. 价值 $\xrightarrow{转化}$ 生产价格
		5. **剩余价值理论意义** 【剥阶必石科】
			1. 揭露资本生产关系剥削本质
			2. 阐明资产阶级无产阶级の阶级斗争经济根源
			3. 支出无产阶级革命の历史必然性
			4. 马克思经济理论基石
			5. 唯物史观+剩余价值 $\Rightarrow$ 社会主义空想到科学
