## 运算与关系性质

| **保留性质**       | 自反  | 反自反 | 对称  | 反对称 | 传递  |
| -------------- | --- | --- | --- | --- | --- |
| 逆 $A^{-1}$     | ✅   | ✅   | ✅   | ✅   | ✅   |
| 交 $A\cap B$    | ✅   | ✅   | ✅   | ✅   | ✅   |
| 并 $A\cup B$    | ✅   | ✅   | ✅   |     |     |
| 差 $A-B$        |     | ✅   | ✅   | ✅   |     |
| 复合 $A \circ B$ | ✅   |     |     |     |     |
## 关系运算性质

| **关系运算** | **笛卡尔积** $A\times B$                             | **关系复合** $A \circ B$                                      | **Warshall 幂** $R^{n}$ |
| -------- | ------------------------------------------------ | --------------------------------------------------------- | ---------------------- |
| 交换律      | ❌                                                | ❌                                                         |                        |
| 结合律      | ❌                                                | $(A\circ B) \circ C = A\circ (B \circ C)$                 |                        |
| 对交分配     | $A\times (B\cap C)=(A\times B) \cap (A\times C)$ | $A \circ (B \cap C) \subset (A \circ B) \cap (A \circ C)$ |                        |
| 对并分配     | $A\times (B\cup C)=(A\times B) \cup (A\times C)$ | $A \circ (B \cup C) = (A \circ B) \cup (A \circ C)$       |                        |
| 对差分配     | $A\times (B\cap C)=(A\times B) \cap (A\times C)$ | $A \circ (B - C) \supset (A \circ B) - (A \circ C)$       |                        |

>[!info] 约定
>表中所有运算的所有分配律都是 **左分配** $\land$ **右分配**的
>例子只写左分配
