## 概论

求解一元高阶同余方程思路

1. 分解模数为素数幂相乘 $$m=p_{1}^{c_{1}}p_{2}^{c_{2}}\dots p_{k}^{c_{k}}$$

2. 解模为素数幂的同余方程 $$f(x)\equiv 0 \ \mathrm{mod}\, p_{i}^{c_{i}}$$

3. 解模为素数的同余方程 $$f(x)\equiv 0 \ \mathrm{mod}\,p$$

4. 用[中国剩余定理 CRT](../%E5%9B%9B%E5%A4%A7%E5%AE%9A%E7%90%86/%E4%B8%AD%E5%9B%BD%E5%89%A9%E4%BD%99%E5%AE%9A%E7%90%86%20CRT.md)合并解 $$\left\{ x \equiv b_{i} \ \mathrm{mod}\,p_{i}^{c_{i}} \right\} \Rightarrow x \equiv b \ \mathrm{mod}\,m $$

## 素数幂模降阶

运用 [Hensel 引理](../Hensel%20%E5%BC%95%E7%90%86.md)

已知
- $f(a)\equiv0 \ \mathrm{mod} \, p^{k}$
- 令 $\lambda := \frac{f(a)}{p^{k}}$

则

- 升阶解 $f(x)\equiv0 \ \mathrm{mod} \, p^{k+1}$ 具有形式 $$x\equiv a+t_{k}\cdot p^{k} \ \mathrm{mod}\,p^{k+1}$$

	- $p \nmid f'(x_{k})$ $\implies$ 唯一解 $$t_{k} = \frac{-f(x_{k})}{p^{k}}\cdot(f'(x_{1})^{-1} \mathrm{mod} \, p) \ \mathrm{mod} \,p$$

	- $p \mid f'(x_{k})$
		- $p \nmid -\frac{f(x_{k})}{p^{k}}$ $\implies$ 无解
		- $p \mid -\frac{f(x_{k})}{p^{k}}$ $\implies$ $p$ 个解 $t_{k}=0,\dots,p-1$

>[!warning] 注意
>注意 $t_{k}$ 的定义式是等于，里面的取模都是算术运算**而非同余运算**
>比如对于 $x^{3}+5x^{2}+9\equiv 0 \ \mathrm{mod}\,3^{2}$ 有 $x_{1} \equiv 1 \ \mathrm{mod}\,3$
>
>那么 $$t_{1}=(-\frac{1^{3}+5\cdot1^{2}+9}{3}\cdot(3\cdot1^{2+10\cdot1 }\% 3)) \% 3$$
>
>得到 $t_{1}=1$
>而不能直接用 $t_{k}\cdot p_{k}=-f(x_{k})\cdot(f^{-1}(x_{1}) \% p)$

## 素数模高阶方程

解

$$
f(x)\equiv 0 \ \mathrm{mod}\,p
$$

1. 【去高阶】用Fermat 小定理 $x^{p-1}\equiv 1 \ \mathrm{mod}\,p$ 简化 $f(x)$，使得 $\mathrm{deg}\,f<p$

>[!warning] 注意
>使用Fermat 小定理时先检查 $x\equiv 0 \ \mathrm{mod}\,p$ 是不是原方程的解！
>否则会漏解哦

2. 【穷举根】用试根法因式分解 $g(x)$ 求得解
	注意灵活运用**系数取模**以及**加减模**同余
