## 定义

正交向量组 orthogonal set :=
- 两两正交

标准正交向量组 orthononormal set :=
- 全是**单位向量** unit vectors
- 两两正交
## 性质

1. **非零** orthogonal set 构成一个子空间的基底 basis
	1. 含**零向量**の正交向量组**线性相关**
	2. 不含零向量の正交向量组线性无关
