- 存储容量
	- 存储容量 = 存储单元个数 $\times$ (存储位元 per 存储单元)
	- 单位
		- b := bits
		- B := Byte = 8b
		- KB := $2^{10}$ B
		- MB := $2^{20}$ B
		- GB := $2^{30}$ B
		- TB := $2^{40}$ B
- 存取时间 $T_{A}$ := 启动一次存储器操作**从开始到完成**的时间
- 存储周期 $T_{S}$ := 连续启动**同一**存储器进行**独立**存取所需**最小间隔**
- 存取宽度 := 每次访存可存取的数据位 **per time**
- 存取带宽 := 每秒传输最大数据量 **per second**
- 可靠性
	- MTBF := Mean Time Between Failure
- 带宽功耗
	- 维持功耗 := 保持时功耗
	- 工作功耗 := 读写时功耗
- 集成度 := 存储容量 per 存储芯片
