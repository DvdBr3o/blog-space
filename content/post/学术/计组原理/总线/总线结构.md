| **总线结构** | **单总线**  | **双总线**        | **三总线**                 |
| -------- | -------- | -------------- | ----------------------- |
| **构成**   | 系统总线     | 系统总线<br>IO 总线  | 系统总线<br>IO 总线<br>DMA 总线 |
| **优点**   | 结构简单 成本低 |                | 吞吐量高                    |
| **缺点**   | 带宽低      | 要增加**通道**等硬件设备 | 同一时刻只能一总线               |

>[!warning] 多通道与多总线
>多通道总线可以提升总线位宽，但多总线不行
