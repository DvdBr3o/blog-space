## 浮点数

IEEE754

![image.png](https://s2.loli.net/2025/07/09/WelH7k486FDQGqv.png)

- $S:=$ 符号位
- $E:=$ 阶码 (移码)
- $M:=$ 尾数 (原码)

`float` $=(-1)^{S}\times 2^{E-127} \times(1.M)$

`double` $=(-1)^{S}\times 2^{E-1023}\times(1.M)$

>[!note] 移码
>已知原码 $e$ 位数为 $k$
>
>则其移码 $E$ 满足 $$E=e+2^{k-1}-1$$
>
>偏置值 $\mathrm{Bias}=2^{k-1}-1$

### 特殊浮点数

| **特殊数类型**    | **阶码** | **尾数** |
| ------------ | ------ | ------ |
| $\pm 0$      | 全 0    | 全 0    |
| $\pm \infty$ | 全 1    | 全 0    |
| 非规格化数        | 全 0    | 非全 0   |
| 非数 NaN       | 全 1    | 非全 0   |
