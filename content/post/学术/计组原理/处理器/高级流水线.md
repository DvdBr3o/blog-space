| **超流水线技术**     | 优点/**原理**           | **缺点**                           |
| -------------- | ------------------- | -------------------------------- |
| 超标量<br>(度=$k$) | 同时发射 $k$ 个指令        |                                  |
| 超长指令字          | 编译器发掘可并行操作，合并成超长指令字 | 1. 需要编译器能力高<br>2. 需要更多硬件资源处理长指令字 |
| 超流水线           | 提高流水线主频             | 频率升高，流水线寄存器开销占比变大，有上限            |
